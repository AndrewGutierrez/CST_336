The Vigenère Cipher is a modified version of the Caesar cipher. Instead of using a number as the key, the Vigenère Cipher uses a word as the key. The letters of the plaintext message are them changed according to the key in the same way they are changed in the Caesar cipher. If the plaintext message is longer than the key, the key is repeated to fit the length of the message.
