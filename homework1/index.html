A cipher is an algorithm that is used to encrypt or decrypt messages. Technology that involves communication, such as phones, use ciphers to encrypt and decrypt messages to maintain a source of security and privacy.

Most ciphers output the same number of characters that was inputted but some ciphers will add letters to the plaintext message before encryption. In order to encrypt the message, a key must be selected and depending on what kind of key is used will determine which kind of cipher is used.
